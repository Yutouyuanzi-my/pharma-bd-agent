- 大卡片 / 侧栏：**20px**

### 阴影
- 卡片：`0 2px 16px rgba(54,44,34,0.06)`（浅淡悬浮阴影）

### 字号 / 字重
| 元素 | 字号 | 字重 |

### 间距栅格
- 基础单位：4 / 8 / 14 / 22 / 28

## 三、核心组件规范

### 1. 导航项（左栏）
- 默认：透明底，文字 `#3d3a36`，16px，居中，圆角 12px，padding 14×16
- Hover：底 `#faf7f2`
- 选中（药丸）：底 `#fbe9dd`，文字 `#1a1816`，700，描边 `#d98841`，圆角 14px

### 2. 指标卡（4 列网格）
- 白底，描边 `#e5e0d8`，圆角 16，阴影同上，padding 20
- 数字 32/800 `#1a1816`；标签 14 `#7a756d`

### 3. 图表
- 横向条形图：轨道底 `#f4f1ec`，填充 `#d98841`，圆角 8
- 竖向柱状图：同上填充，圆角 `8 8 0 0`
- 折线图：线 `#d98841` 2.5px，面积填充 `rgba(217,136,65,0.10)`

### 4. 试验卡片
- 白底，描边，圆角 14，阴影；标题 15/800，元信息 13 `#7a756d`
- 状态 chip：`#fbe9dd` 底 `#8b5020` 字，圆角全圆

### 5. 按钮
- 主按钮：`#d98841` 底白字，10px 圆角，12×22 padding
- 次按钮（软）：`#fbe9dd` 底 `#8b5020` 字，描边 `#f0cfae`

### 6. Tabs / 表格
- Tab：默认白底描边，选中 `#fbe9dd` 底
- 表格：1px `#e6e0d6` 分隔，表头 `#faf7f2` 底 800

## 四、交互说明（原型可点击逻辑）

1. **左导航 7 项点击** → 切换中间视图（总览 / 竞争格局 / 试验检索 / 每日监测 / 竞品对比 / 中国管线 / 智能助手）
2. **右栏「快捷查询」4 按钮** → 跳转「智能助手」并预填查询文本（如"分析 NSCLC 领域的竞争格局，并列出近期重要更新"）
3. **「中国管线」内 tabs（管线检索 / CDE 审批）** → 切换两个面板
4. **「生成看板 / 检索 / 监测 / 对比」类主按钮** → 触发加载态（spinner）→ 渲染数据态（原型默认展示数据态，便于截图）

## 五、在 Figma / 墨刀 中的制作步骤

1. 新建桌面画板 **1280×800**（按 13 个画板复制）
2. 建立全局 Style：颜色变量（上表）+ 文字样式 + 圆角 / 阴影
3. 画「组件库」：导航项(3 态) / 指标卡 / 条形图 / 柱状图 / 折线图 / 试验卡 / 按钮 / tabs / 表格 / chip
4. 左栏：品牌区 + 分割线 + 7 导航项（复制组件）
5. 右栏：标题 + 3 段（快捷查询 / 我的监测 / 最近查询）各加分割线
6. 中栏逐页填充（指标卡 + 图表 + 列表 / 表格）
7. 用「原型」模式连线：导航 → 对应页；快捷查询 → 智能助手页并设文本框默认值
8. 导出 PDF / 录屏，用于演示与交付
